Performance Feedback Punching Bag System (PFPB) - Patent Pending

PFPB was myself, and two collegues Senior Design Project while at Western Michigan University. We developed a auditory feedback workout system for the Psycology Department to run behavioral tests with. The system consisted of a microcontroller box, that would mount to the top of a punching bag stand, connected to an array of removable accelerometers on the punching bag itself. The microcontroller then communicted with a custom Java app we also wrote to track workout sessions over time, playback user selected music playlists, and change the frequency & speed of the playback based on the input from the microcontroller and the user defined goals for their workout. The device was so well recieved that the OVPR department at Western Michigan University filed for a patent on the device as well. Feel free to checkout the source code as it has now been open sourced.
